# 7.2. IO, Network Sockets

Java: `java.io.*`, `java.net.*` packages

Vala: `GLib` namespace (imported by default), `--pkg gio-2.0`, GIO is part of GLib

See [GIO Examples](https://wiki.gnome.org/Projects/Vala/GIOSamples), [GIO Networking Examples](https://wiki.gnome.org/Projects/Vala/GIONetworkingSample) on the wiki.
