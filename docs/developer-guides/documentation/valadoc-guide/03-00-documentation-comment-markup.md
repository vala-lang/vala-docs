# 3. Documentation Comment Markup

## 3.1. Documentation Comment Structure

```vala
/**
 * [brief description]
 *
 * [long description]
 *
 * [taglets]
 */
```

::: info Note

When no comments are present, Valadoc will only extract symbols to use
in the documentation.
:::

documentation comment.
