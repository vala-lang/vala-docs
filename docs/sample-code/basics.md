# Basics: Collections, Files, I/O, Networking, IPC samples

These examples cover libgee collections, GIO file and stream I/O, compression and
application settings, TCP/UDP networking with GIO, D-Bus clients and servers, and
Vala's `async` / `yield` patterns for asynchronous APIs. They adapt material that
previously lived on the GNOME Wiki Vala project (now retired). This section is
the maintained home for those samples.
