# Language Features and Introductory Samples

These examples introduce core Vala language features: console programs, strings,
Unicode, GObject properties and signals, and conditional compilation. They adapt
material that previously lived on the GNOME Wiki Vala project (now retired).
This section is the maintained home for those introductory samples.
